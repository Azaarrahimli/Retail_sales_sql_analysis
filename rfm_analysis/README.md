## Overview

This project applies RFM (Recency, Frequency, Monetary) analysis to segment customers based on purchasing behavior. The objective is to identify high-value, regular, and at-risk customers.

### Tools:

Oracle SQL

Power BI

### Dashboard Includes:

Total Sales

Total Customers

VIP Customers and VIP %

Customer distribution by segment

Revenue distribution by segment

Filters by Year and Region
